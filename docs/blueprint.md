# **App Name**: RivalXP

## Core Features:

- AI Rival Personalization & Taunts: Users customize their AI Rival's appearance (sprite) and choose a personality tone. The AI acts as a tool, generating light, contextual taunts throughout the day, dynamically responding to user activity (or inactivity) based on the chosen personality.
- Task Creation & Management: Users define and manage daily quests with titles, duration/completion types, and XP rewards, limited to timed and binary tasks as MVP constraint.
- Real-Time XP Race Dashboard: A dashboard prominently displaying user and rival XP in real-time. Includes customizable pixel art sprites with animations for XP bursts on task completion and visual progression for the rival during user inactivity.
- Focus Mode Mechanic: Users can activate a 'Focus Mode' during active work sessions, which pauses or significantly slows the rival's XP accumulation, providing control and mitigating hopelessness.
- Daily Cycle & Summary: Manages the daily XP race, ensuring it concludes at 11:59 PM, and presents an end-of-day summary with winner animations and performance statistics.
- Progress Persistence: Securely stores all user data, including rival customizations, created tasks, daily progress, and overall XP accumulation to maintain game state across sessions.
